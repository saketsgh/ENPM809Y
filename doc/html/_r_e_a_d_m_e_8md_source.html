7 Scenario:
8 ========
9 A robot navigates through a maze to reach the center of the maze.
10 
11 Objectives:
12 ========
13 - Write a program that generates a path from the current position (S) of a robot in a maze to the center of
14 the maze (G).
15 - The robot that navigates the maze will always be a robot from a derived C ++ class
16 - Once a path is generated, the robot is tasked to follow the path.
17 - The micromouse simulator (https://github.com/mackorone/mms) will be used to display the result on
18 the screen.
19 - The program exits when either the robot reaches G or there is no solution from S to G.
20 
21 Maze:
22 =======
23 
24 - All mazes have the same dimensions 16 × 16 cells
25 - All mazes have walls around their perimeter.
26 - The robot has no knowledge of walls (even the ones around the perimeter) beforehand.
27 - Walls are discovered only when the robot drives through cells that have walls.
28 - The coordinate system for all the mazes is displayed in Figure 2.
29  - With x ∈ [0, 15] and y ∈ [0, 15]
30 - Any of the 4 cells that are in the center of the maze constitutes the goal (G) the robot needs to reach. The coordinates of the goal cells are consistent among
31 mazes:
32  – (7,7)
33  – (7,8)
34  – (8,7)
35  – (8,8)
36 • When the simulation starts, the robot will always start at position (0,0) and will always face north.

39 To run the code:
40 ========
41  1. Navigate to mms/bin
42  2. Run the following command: './mms'
43  3. Choose one of the mazes available in mazefiles folder.
44  4. Select Final_Project/src as the code directory for the algorithm and run the simulator.
45  5. Include this line as build command
46 
47  ```g++ ./src/API/api.h ./src/API/api.cpp ./src/Maze/maze.h ./src/Maze/maze.cpp ./src/Algorithm/algorithm.h ./src/Algorithm/algorithm.cpp ./src/LandBasedRobot/landbasedrobot.h ./src/LandBasedRobot/landbasedrobot.cpp ./src/LandBasedWheeled/landbasedwheeled.h ./src/LandBasedWheeled/landbasedwheeled.cpp ./main.cpp```
48 
49  6. The implementation provided has two types of algorithms:
50 
51  --> fp::Algorithm::solve(fp::LandBasedRobot *robot_) ==> Plans the path using DFS and checks for walls while moving along that path. (Psuedo code given in document)
52 
53  --> fp::Algorithm::solve_V2(fp::LandBasedRobot *robot_) ==> Updates all the walls of the cell the robot is in and then starts exploring using DFS.
