DESCRIPTION

This project generates a path from the current position(S) of a robot in a maze to the center of the maze(G). Once a path is generatedm the robot is tasked to follow the path. The micromouse simulator is used to display the result on the screen. The prorgam exits when either the robot reaches G or there is no solution from S to G.
